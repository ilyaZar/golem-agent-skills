# Golem Project Guidance

This project is a golem-based Shiny application.

Use the project-local golem skills shipped with this project when changing app
structure, modules, resources, development scripts, deployment files, or tests.

Relevant local skills:
- `golem-create-golem`
- `golem-add-module`
- `golem-add-function`
- `golem-check-app`
- `golem-run-tests`
- `golem-fix-missing-ns`
- `golem-fix-missing-ns-colin`
- `golem-upgrade`
